# Streamlit-приложение для сбора отзывов по ЖК конкурентов с Яндекс Карт

## Что делает приложение

Приложение позволяет:

1. загрузить список жилых комплексов;
2. выполнить поиск карточек на Яндекс Картах;
3. вручную подтвердить корректные карточки;
4. собрать до 100 последних отзывов по каждой подтвержденной карточке;
5. скачать результат в CSV.

---

## Стек

- Python 3.11+
- Streamlit
- Playwright for Python
