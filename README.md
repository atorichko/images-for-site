# Yandex Maps Reviews Scraper for Residential Complexes

Консольный Python-скрипт для поиска карточек жилых комплексов на Яндекс Картах и сбора последних отзывов по подтвержденным карточкам.

## Что делает скрипт

1. Принимает список ЖК:
   - построчно из консоли
   - или из `.txt` файла
2. Для каждого ЖК выполняет поиск на Яндекс Картах
3. Показывает найденную карточку:
   - исходное название ЖК
   - найденное название карточки
   - адрес
   - ссылку
4. Позволяет пользователю:
   - подтвердить карточку
   - изменить запрос и повторить поиск
   - исключить ЖК
5. После подтверждения списка:
   - собирает до 100 последних отзывов по каждой карточке
6. Сохраняет результат в CSV:
   - кодировка `utf-8-sig`
   - разделитель `;`
